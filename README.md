# customer-data
data reading from file

## Project Setup

- git clone https://github.com/StalJoku/customer-data.git CustomerData
- run command: docker compose up -d

### Web Address

url http://127.0.0.1:8080/

### Configuration and Usage

open src/.env file and set which file formats or file names program will be reading. Go to url http://127.0.0.1:8080/
upload your file and document content will print on screen
